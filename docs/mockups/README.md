## 設計の芯（両バージョン共通）
- **楽しく**：進捗メタファー（娘=満月ゲージ／ママ=おかしびん）・10個で1スタンプ・図鑑コレクション。
- **ちょっと頑張れる**：あと◯個・1日1チャレンジ・やわらか連続。
- **せめない／ねぎらう**：できてない＝静かな色、いつでも取消、減点なし、質問は1つだけ（要件8・10・12・17・18）。
