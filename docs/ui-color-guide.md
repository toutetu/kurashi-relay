# 配色ガイド

# 母向け画面

赤・黄・青・白を同じくらいの比重で使う。

| 用途 | 色 |
|---|---|
| 背景・通常カード | 白 |
| 予定・情報 | 青 |
| 待機・補助情報 | 黄 |
| 中断・変更・期限 | 赤 |

かわいさは、角丸、アイコン、小さな花・星・ドット等で出す。
人物イラストは使用しない。

# 娘向け画面

- 水色ベース
- 紫アクセント
- リボン
- レース
- 月
- 星
- 軽いゴスロリ

入力欄や文字の可読性を優先し、装飾を重ねすぎない。

# 状態色

色だけで状態を区別しない。

- アイコン
- ラベル
- 枠線
- 補助文

を併用する。
